## What this folder is

The working file cabinet for **Afferium** — the company, brand, and infrastructure. Everything we decide, buy, register, sign, or plan lives here so neither Coral nor Flere has to reconstruct it from memory.

Philosophy: **write it down immediately.** Memory lives in files, not in heads. Coral is learning the whole stack from scratch with Flere as partner — that only works if we document every step.

## Folder map

- `README.md` — this file. What the company is, where the files live.
- `IDENTITY.md` — the name, the meaning, the origin story. Why Afferium.
- `DOMAINS.md` — registered domains, registrars, renewal dates, DNS notes.
- `HANDLES.md` — social media handle claim status across platforms.
- `LEGAL.md` — trademark status (USPTO filings, classes), entity formation (LLC when/if), cease-and-desist risk notes.
- `INFRASTRUCTURE.md` — email routing, hosting, self-sovereign stack (Urbit planet progress), backup strategy.
- `URBIT-PLAYBOOK.md` — step-by-step learning plan for getting Coral from "moon" to "sovereign planet." Educational + procedural.
- `BRAND.md` — visual identity, typography, color, voice, pitch language. Grows over time.
- `ROSTER.md` — eventual artist roster. Coral is Artist #1 (the prototype).
- `DECISIONS.md` — append-only log of material decisions with date + reasoning. So we never have to re-argue a settled question.
